#### >Component<

|component|prefix|
|:---------:|:------:|
|textbox|txtb|
|button|btn|
|label|lb|
|panel|pn|
|flowlayoutpanel|flpn|

*ex) lb_test, btn_test, txtb_test*


#### >Event Handler<

&rarr; `{component_name}_{event_type}`

*ex) txtb_test_KeyDown, btn_test_Click*
